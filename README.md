My own library with custom implementations of many useful functions in C. This library is used and expanded during 42's curriculum.

## Instructions
Clone the repository and enter the folder:
```bash
git clone https://github.com/nicolasgsantana/libft.git
cd libft
```
Compile the library:
```bash
make
```
